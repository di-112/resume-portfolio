Роль:

Frontend-разработчик

Стек:

React.JS / ES6, TypeScript, Webpack, Eslint, Less, Css Module, Ant Design, DevExtreme, d3, MobX, Redux, Storybook, Figma

Описание:

Проект является React приложением, реализованным через микрофронтенды, для ведения аналитики и статистики по технологическим процессам. Каждая часть приложения собирается через webpack. Используется собственная ui библиотека, представленная в Storybook. Все компоненты были переписаны с нуля без использования сторонних библиотек. Главным исключением является компонент таблицы, построенный на базе DevExtreme. Основные требования проекта - создание ведомостей, сравнение различных анализов качества. Также работа со справочниками - добавление новых, редактирование и удаление. Осуществил правки текущего функционала проекта. Реализовал отдельные функциональные блоки. Разработал графики. Создал сложные, многофункциональные компоненты, внедрил их в функциональные формы. Разработал с нуля большую часть компонентов для ui-библиотеки (DateTimePicker, Tabs, ModalWindow и другие). Кастомизировал таблицу из DevExtreme.

Приложение на Electron JS для планирования рабочего времени

Команда:

Full Stack - разработчик Аналитик

Роль:

Full Stack разработчик

Стек:

ElectronJS, ReactJS, TypeScript, MobX, ChakraUI

Описание:

Десктопное приложение для создания отчетов о трудозатратах на проектах и формирования уведомлений о начале/ завершении рабочего дня. UI реализован с помощью Chakra. Frontend собирается через сборщик vite. Для создания установочного файла под любые OC используется electron-builder. Пользователю предоставляется возможность параллельно авторизоваться в Redmine, Jira и Gitlab. Для каждого трекера есть возможность запрашивать активные задачи и и тредозатраты за день для формирования отчетов. В случае, если пользователь не сформировал какой-либо из отчетов за день, через установленное время появляется соответствующее уведомление в OC На проекте реализовал UI часть, добавил интеграцию React приложения с ElectornJS. Создал механизмы взаимодействия с трекером Redmine, Jira и Gitlab.
